Мы не используем в этом пакете энамы, т.к. значения переменных должны быть константами не только в runtime, как это происходит в случае с энам, а должны инициализироваться (стать константами) в статических блоках до runtime, уже при компиляции.
Это необходимо для того, чтобы мы могли подставлять значения в аннотации, в том числе кафки. Параметры аннотаций требуют инициализации до runtime.
